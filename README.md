# Простой WB парсер :)

## Что это?
Скрипт для автоматического сбора данных о товарах с маркетплейса Wildberries.
Основные параметры вынесены в файл  wb_settings внутри каталога wb_parser_configs

## Основные параметры внутри файла wb_settings:

 - WB_HOME_URL - стартовая страница [лучше не менять так как все под нее вшито]
 - SEARCH_TEXT - поисковый запрос [этот текст будет вбиваться в строку поиска сайта]
 - MAX_PRODUCT_FOR_PARSSING - число продуктов которые будет брать [0 - будет брать товары до встроенного ограничения]
 - OUTPUT_FILE - файл куда записываем результат
 - HEADLESS_MODE - запуск скрипта в фоновом режиме

## Преборазвоние в xlsx:
Для конвертации результата работы в xls запустить csv_filter.py.
Он выполнит конвертацию .csv документа в .xlsx.

!! Важно: сейчас он просто работает с заранее заготовленными именами -> имена файлов не менять для корректной работы !!

## Требования (то, на чем должно работать):
 - Python: 3.14
 - Браузер: GoogleChrome
 - Взять зависимости из requirements.txt
